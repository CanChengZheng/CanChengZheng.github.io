Android 开发如何结束多个 Activity

发表于 2017-10-23

问题背景

  • 在Android应用程序运行的过程中，可能会遇到需要一次性结束掉一部分或者所有 Activity的情况，这篇文章整理了网络上的一些方法。

解决方案

  1. 创建一个专门用于管理 Activity 的类
    1. 在类中实现响应的接口方法，如addActivity(Activity)、delActivity(Activity)、finishAll()等等。
    2. 在类中使用list保存Activity的对象
    3. 由于程序中有多个Activity，调用addActivity()的代码需要重复写多次，因此可以写一个的基类BaseActivity，在onCreate()或者onResume()中把当前的Activity加入到栈中，每一个Activity都继承至BaseActivity
    4. 当需要结束掉所有Activity，可以调用Activity管理类中的finishAll()方法，对所有Activity调用finish()
  2. 如果是结束部分Activity的情况，可以考虑利用Activity的四种启动模式来进行操作
    1. 利用启动方式singleTask，当已经 Activity 返回栈中存在 Activity A，使用singleTask，系统会首先在返回栈中检查是否存在该Activity的实例，如果发现已经存在，则直接使用该实例，并把返回栈中在该Activity上方的所有Activity出栈，从而达到关闭部分Activity的效果。
  3. 使用广播，每个Activity中收到这个广播，自己将自己finish掉。
